- Consensus bullet chart margins: l=160px, r=40px, t=40px, b=40px (Plotly layout margin — pixel-exact values required by Plotly API, not CSS spacing)

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Display | 28px | 600 | 1.2 | Hero KPI value (e.g., "$847B", "23.4%") |
| Heading | 20px | 600 | 1.2 | Card titles, section headings, tab panel titles, header title |
| Body | 16px | 400 | 1.5 | Narrative text, card descriptions, insight text |
| Label | 12px | 400 | 1.4 | KPI sub-labels, KPI scope labels, vintage footers, attribution footnotes, chart axis tick labels, tab navigation (unselected and selected) |

Notes:
- KPI label above value: 12px weight 400, color #888 — matches existing `ATTRIBUTION_STYLE` and `FONT_LABEL`
- KPI sub-label below value (scope, uncertainty range): 12px weight 400, color #999 — same size as Label role, lighter color creates visual hierarchy without a fifth size
- Vintage footer text: 12px weight 400, color #AAAAAA — lighter color than label so it recedes; same size tier as Label
- Tab navigation: 12px weight 400 unselected, 12px weight 600 selected — collapsed from previous 14px to stay within 4-size scale

**Revised from draft:** Removed 11px and 14px sizes. Sub-labels and vintage footers now use 12px (Label tier) with lighter color values (#999 and #AAAAAA respectively) to maintain visual hierarchy without additional size steps. Tab navigation collapsed from 14px to 12px. Weight 700 replaced by 600 — only two weights declared: 400 (regular) and 600 (semibold).

**Source:** Detected from `src/dashboard/charts/styles.py` (`FONT_DISPLAY`, `FONT_HEADING`, `FONT_BODY`, `FONT_LABEL`) and `src/dashboard/layout.py`. Display size 28px for card-bound KPI numerics (36px is chart-level only).


## Interaction Contract

### Confidence Traffic Light

- Rendered as `html.Span("●", style={"color": confidence_color})` — 14px, verticalAlign middle
- Computed at render time from CI width ratio (see Color section for thresholds)
- No interactivity — read-only indicator, no tooltip on dot itself

### Consensus Bullet Chart Marker

- Inside range (p25 ≤ model ≤ p75): marker color #2ECC71 (green)
- Outside range: marker color #F39C12 (amber)
- Tooltip on hover: "Model: ${X}B vs Consensus: ${Y}–${Z}B — divergence: +N%"
- No click behavior — display only
- Segments with `estimated_flag=True` anchors: show "No consensus data" text row, not a bar

### Tab Navigation

- Basic tab is the first tab and the default (`value="basic"` on `dcc.Tabs`)
- Basic tab ignores the mode-toggle entirely — `build_basic_layout()` takes `mode` parameter but does not branch on it
- Existing tabs (Overview, Segments, Drivers, Diagnostics) unaffected by Basic tab addition, except they must have the `elif active_tab == "basic"` guard added before them in `callbacks.py`

### Vintage Footer

- Not interactive — static text only
- Placement: `html.P` below the last `dcc.Graph` in each section, text-align right
- Color: #AAAAAA (lighter than chart axis labels to visually recede)
- Margin-top: 8px (sm token)

### Mode Toggle Interaction (existing, extended)

- Basic tab: mode toggle visible in header but ignored by Basic tab
- Normal mode: shows consensus panel + revenue multiples table in Overview tab
- Expert mode: shows full divergence rationale text (8 analyst firms listed) in Overview tab; replaces PCA expert card with v1.1 anchor model explanation

## Accessibility Notes

- Confidence colors are not the only encoding — the ● symbol is always present; a screen reader reads "●" as a bullet. Consider adding `aria-label=f"Confidence: {level}"` to the Span element.
- Consensus bullet chart: Plotly's hover tooltip provides text equivalent of the visual range encoding.
- All numbers include units in their label or sub-label text (e.g., "Nominal USD", "USD Billions").
- Target: WCAG AA contrast for all body text. #888888 on #FFFFFF = 3.5:1 (AA for large text, borderline for body — acceptable given this is supplementary label text, not primary content).
