Create `scripts/run_statistical_pipeline.py` that does the following:

1. **Generate representative synthetic data** for all 4 AI segments. The script must work without live API data. Create a DataFrame matching PROCESSED_SCHEMA format with columns: `year`, `value_real_2020`, `industry_segment`. Generate 15 years of data (2010-2024) per segment with:
   - A realistic upward trend (different growth rates per segment)
   - A structural break at 2022 (step increase in growth rate to simulate GenAI surge)
   - Random noise via numpy (seed=42 for reproducibility)
   - Segments: ai_hardware, ai_infrastructure, ai_software, ai_adoption

2. **For each of the 4 segments**, run the full ARIMA vs Prophet comparison pipeline:
   a. Extract the segment series (groupby year, sum value_real_2020, sort by year)
   b. Call `select_arima_order(series)` to get (p,d,q) via AICc
   c. Call `run_arima_cv(series, order, n_splits=3)` for ARIMA CV metrics
   d. Call `run_prophet_cv(df, segment, n_splits=3)` for Prophet CV metrics (pass the full long-format df, not just the series)
   e. Call `compare_models(arima_cv, prophet_cv, segment)` to determine winner
   f. Based on winner, extract residuals:
      - If ARIMA wins: `fit_arima_segment(series, order)` then `get_arima_residuals(results, series.index)`
      - If Prophet wins: `fit_prophet_segment(df, segment)` then `get_prophet_residuals(model, prepared_df)` where prepared_df is the ds/y format DataFrame
   g. Store (residual_series, winner_name) in segment_residuals dict

3. **Call `save_all_residuals(segment_residuals, output_path)`** where output_path is `str(DATA_PROCESSED / "residuals_statistical.parquet")`.

4. **Print a summary table** to stdout showing per-segment: ARIMA RMSE, Prophet RMSE, winner, residual count.

5. **Add `if __name__ == "__main__":`** guard. Suppress Prophet/cmdstanpy logging at WARNING level.
