    3. Create `src/models/statistical/regression.py` with:

    **fit_top_down_ols_with_upgrade(y: np.ndarray, X: np.ndarray) -> tuple:**
    - Step 1: ols_res = OLS(y, X).fit()
    - Step 2: Breusch-Pagan test — bp_stat, bp_pval, _, _ = het_breuschpagan(ols_res.resid, X)
    - Step 3: Ljung-Box test — lb_result = acorr_ljungbox(ols_res.resid, lags=[1], return_df=True); lb_pval = lb_result["lb_pvalue"].iloc[0]
    - If bp_pval < 0.05: weights = 1.0 / (ols_res.fittedvalues**2 + 1e-8); final_res = WLS(y, X, weights=weights).fit(); model_type = f"WLS (heteroscedasticity detected, BP p={bp_pval:.3f})"
    - Elif lb_pval < 0.05: final_res = GLSAR(y, X, rho=1).iterative_fit(maxiter=10); model_type = f"GLSAR (autocorrelation detected, LB p={lb_pval:.3f})"
    - Else: final_res = ols_res; model_type = "OLS (no heteroscedasticity or autocorrelation detected)"
    - diagnostics = {"bp_stat": bp_stat, "bp_pval": bp_pval, "lb_pval": lb_pval, "r2": ols_res.rsquared, "r2_adj": ols_res.rsquared_adj}
    - Return (final_res, model_type, diagnostics)

    **temporal_cv_generic(series: np.ndarray, fit_fn: callable, forecast_fn: callable, n_splits: int = 3) -> list[dict]:**
    - Uses sklearn.model_selection.TimeSeriesSplit(n_splits=n_splits)
    - For each fold: train, test = series[train_idx], series[test_idx]
    - fitted = fit_fn(train)
    - forecast = forecast_fn(fitted, steps=len(test))
    - rmse = np.sqrt(np.mean((test - forecast)**2))
    - mape = np.mean(np.abs((test - forecast) / (test + 1e-10))) * 100
    - Append {"fold": i, "train_end": train_idx[-1], "test_end": test_idx[-1], "rmse": rmse, "mape": mape}
    - Return list of fold results

    Imports:
    - `from statsmodels.regression.linear_model import OLS, WLS, GLSAR`
    - `from statsmodels.stats.diagnostic import het_breuschpagan, acorr_ljungbox`
    - `from sklearn.model_selection import TimeSeriesSplit`
    - `import numpy as np`
