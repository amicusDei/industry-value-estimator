    **run_chow(series: pd.Series, break_idx: int) -> dict:**
    - Manual SSR-based Chow test (no built-in statsmodels function)
    - X = np.column_stack([np.ones(len(y)), np.arange(len(y))])
    - Compute SSR_full, SSR_pre ([:break_idx]), SSR_post ([break_idx:])

    - p_value = 1 - scipy.stats.f.cdf(F, dfn=k, dfd=n-2*k)
    - Returns {"F_stat": float, "p_value": float, "break_year": series.index[break_idx]}

    Create `src/diagnostics/model_eval.py` with these functions:

    **compute_rmse(actual: np.ndarray, predicted: np.ndarray) -> float:**
    - `np.sqrt(np.mean((actual - predicted) ** 2))`

    **compute_mape(actual: np.ndarray, predicted: np.ndarray) -> float:**
    - `np.mean(np.abs((actual - predicted) / actual)) * 100`
    - Guard: raise ValueError if any actual value is zero

    **compute_r2(actual: np.ndarray, predicted: np.ndarray) -> float:**
    - `1 - np.sum((actual - predicted)**2) / np.sum((actual - np.mean(actual))**2)`

    **compute_aic_bic(residuals: np.ndarray, n_params: int) -> dict:**
    - n = len(residuals)
    - sse = np.sum(residuals**2)
    - aic = n * np.log(sse/n) + 2 * n_params
    - bic = n * np.log(sse/n) + n_params * np.log(n)
    - aicc = aic + (2 * n_params * (n_params + 1)) / (n - n_params - 1)
    - Return {"aic": aic, "bic": bic, "aicc": aicc}

    **ljung_box_test(residuals: np.ndarray, lags: int = 1) -> dict:**
    - Use `statsmodels.stats.diagnostic.acorr_ljungbox(residuals, lags=[lags], return_df=True)`
    - Return {"statistic": float, "p_value": float}

    **compare_models(arima_cv: list[dict], prophet_cv: list[dict], segment: str) -> dict:**
    - Compute mean RMSE and mean MAPE for each model from CV fold results
    - winner = "ARIMA" if arima_mean_rmse <= prophet_mean_rmse else "Prophet"
    - Return {"segment": segment, "arima_mean_cv_rmse": float, "prophet_mean_cv_rmse": float, "arima_mean_cv_mape": float, "prophet_mean_cv_mape": float, "winner": str, "margin_pct": float}
