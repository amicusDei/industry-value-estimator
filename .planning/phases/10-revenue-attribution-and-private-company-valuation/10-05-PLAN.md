Three sub-tasks in sequence:

**A. Fix circular MAPE for soft actuals in walk_forward.py:**

The root cause: `forecasts_ensemble.parquet` point estimates for 2023-2024 are identical to `market_anchors_ai.parquet` medians because the Phase 9 model was calibrated to those medians. Comparing them produces MAPE=0.0 which is meaningless.

Fix approach — for soft actuals, acknowledge the circularity explicitly:
1. In `run_walk_forward()`, after computing MAPE for soft actuals, add a `circular_flag` column. Set `circular_flag=True` when `abs(actual_usd - predicted_usd) < 0.01` (i.e., values are effectively identical).
2. When `circular_flag=True`, set `mape_label="circular_not_validated"` instead of "acceptable" — this makes the limitation visible in the output rather than hiding behind a misleadingly good score.
3. Add a docstring note explaining: "Soft actual MAPE is circular when the model was calibrated against the same market anchor medians used as soft actuals. Hard actuals (EDGAR direct-disclosure) are the primary non-circular validation signal."

This is the honest approach per the project's core value ("produce valuations that people can actually trust"). Do NOT try to "fix" the soft MAPE by using a held-out analyst source — there is only one market anchors dataset. The fix is transparency, not data gymnastics.

**B. Handle 2022 fold absence:**

`forecasts_ensemble.parquet` starts at year 2023 — there are no 2022 rows. The current code already handles this gracefully (prints a skip message). Formalize this:
1. In `walk_forward.py`, update the module docstring to say "2 evaluation folds (2023, 2024)" instead of "3 evaluation folds". Keep `EVALUATION_YEARS = [2022, 2023, 2024]` as-is — the code correctly skips 2022 when no forecast data exists.
2. Add a constant `MIN_FOLDS = 2` and after the walk-forward loop, if `len(unique_years_evaluated) < MIN_FOLDS`, print a warning but still return results.
3. In the results summary print, add: "NOTE: 2022 fold absent — forecasts_ensemble.parquet starts at 2023. {N} of 3 possible folds evaluated."
