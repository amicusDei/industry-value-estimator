**Extend src/ingestion/market_anchors.py** with a new top-level function `compile_and_write_market_anchors` that performs the full pipeline:

```python
def compile_and_write_market_anchors(industry_id: str = "ai") -> Path:
    """
    Full market anchors pipeline: YAML → compile → deflate → interpolate → validate → Parquet.

    Steps:
    1. Call compile_market_anchors(industry_id) to get nominal compiled DataFrame
    2. Load GDP deflator data from data/processed/world_bank_{industry_id}.parquet
       - Read the existing World Bank Parquet, extract the gdp_deflator_index column
       - Need year + deflator value for the "global" economy (or US as proxy)
    3. For each _nominal column (p25, median, p75), apply deflate_to_base_year()
       to produce corresponding _real_2020 columns
    4. Interpolate gaps: for each segment, ensure years 2017-2025 are present.
       Missing years filled via linear interpolation with estimated_flag=True.
       Use pandas interpolate(method='linear') on sorted year index per segment.
    5. Validate against MARKET_ANCHOR_SCHEMA (update schema to include _real_2020 columns)
    6. Write to data/processed/market_anchors_{industry_id}.parquet with provenance metadata:
       - b"source": b"market_anchors"
       - b"industry": industry_id
       - b"fetched_at": UTC ISO timestamp
       - b"registry_path": path to YAML
       - b"reconciliation_method": b"scope_normalized_median"
    7. Return Path to written Parquet file
    """
```

Implementation details for deflation:
- Read `data/processed/world_bank_{industry_id}.parquet` to get GDP deflator
- Extract rows where economy=="USA" (US deflator as proxy for global USD deflation)
- Build a Series with year as index, deflator value as values
- For each nominal column, call `deflate_to_base_year(nominal_series, deflator_series)`
- Name output columns: `p25_usd_billions_real_2020`, `median_usd_billions_real_2020`, `p75_usd_billions_real_2020`

Implementation details for interpolation:
- Group by segment
- For each segment, reindex to full year range 2017-2025
- Use `df.interpolate(method='linear')` for numeric columns
- Set `estimated_flag=True` for any row that was originally missing (NaN before interpolation)
- Set `n_sources=0` and `source_list=""` for interpolated rows
