Implementation details:
- `fetch_company_filings` creates a `Company(cik)` object, calls `company.get_filings(form=form_type)` for each form type, filters by date range `f"{start_year}-01-01:{end_year}-12-31"`, then for each filing calls `filing.xbrl()` and queries concepts in XBRL_CONCEPTS priority order (first non-null wins)
- Each row gets `bundled_flag = cik in BUNDLED_SEGMENT_COMPANIES`
- `fetch_all_edgar_companies` reads `config["edgar_companies"]`, defaults form_types to `["10-K", "10-Q"]` unless the company entry has `form_types` key (for 20-F filers like TSMC/Accenture)
- Start year 2020, end year 2024 (from CONTEXT.md: 5 years)
- Wrap each company fetch in try/except and log errors — do not abort on single company failure
- `save_raw_edgar` writes to `DATA_RAW / "edgar" / f"edgar_{industry_id}_raw.parquet"` with provenance metadata using `pa.Table.replace_schema_metadata()` pattern from world_bank.py
