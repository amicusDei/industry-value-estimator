       - Same SEGMENTS list: `["ai_hardware", "ai_infrastructure", "ai_software", "ai_adoption"]`

       - `run_pipeline()` function with these steps:
         a. Load residuals: `pd.read_parquet(DATA_PROCESSED / "residuals_statistical.parquet")`.
            If file doesn't exist, run `run_statistical_pipeline.py` first (print message and sys.exit(1)).
         b. Get data vintage: `get_data_vintage(residuals_df)`.
         c. Build features: `build_residual_features(residuals_df)`.

         d. Per-segment loop:
            - Filter features for segment.
            - Extract X (FEATURE_COLS columns as numpy) and y (residual column as numpy).
            - Run LightGBM CV: `lgbm_cv_for_segment(y, X, n_splits=3)`.
            - Compute mean LightGBM CV RMSE from fold results.
            - Get statistical CV RMSE: Re-run or hardcode from Phase 2 output. For synthetic data, use a placeholder approach: compute statistical model's in-sample RMSE on the same residuals (since statistical model residuals are the LightGBM target, the statistical "forecast" is zero — RMSE = std(residuals)). This gives a valid baseline comparison.
            - Compute ensemble weights: `compute_ensemble_weights(stat_rmse, lgbm_rmse)`.
            - Fit point model: `fit_lgbm_point(X, y)`.
            - Fit quantile models: `fit_all_quantile_models(X, y)`.
            - Generate forecast features for 2025-2030: Build future feature rows with year_norm = (year - 2010) / 14.0, residual_lag1 and residual_lag2 extrapolated from last known residuals (use last 2 residuals as constant forward projection).
            - Predict: point model and all 4 quantile models on future features.
            - Blend: `blend_forecasts(stat_baseline=0.0, lgbm_correction=point_pred, stat_weight, lgbm_weight)` — for the forecast horizon, stat_baseline is the statistical model's forecast (use 0.0 as the residual correction is the full signal).
            - Build segment forecast dict: historical years (2010-2024) with actual values + forecast years (2025-2030) with predicted values.
            - Compute SHAP: `compute_shap_values(point_model, X_df, FEATURE_COLS)`.

         e. Serialize models:
            - `joblib.dump(model, MODELS_DIR / "ai_industry" / f"{segment}_lgbm_{key}.joblib")` for each model (point + 4 quantiles).
            - `joblib.dump(weights_dict, MODELS_DIR / "ai_industry" / "ensemble_weights.joblib")` for all segment weights.

         f. Save SHAP summary plot: `save_shap_summary_plot(shap_dict, X_df, MODELS_DIR / "ai_industry" / "shap_summary.png")`.

         g. Build and save forecast DataFrame:
            - `build_forecast_dataframe(all_segment_forecasts, vintage)`.
            - `df.to_parquet(DATA_PROCESSED / "forecasts_ensemble.parquet", index=False)`.

         h. Print summary table (segment, stat_rmse, lgbm_rmse, stat_weight, lgbm_weight).

       - `if __name__ == "__main__": run_pipeline()`.
