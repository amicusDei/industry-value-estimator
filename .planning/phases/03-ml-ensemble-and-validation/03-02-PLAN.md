           row["ci80_upper"] = max(row["ci80_upper"], row["point_estimate_real_2020"])
           row["ci95_upper"] = max(row["ci95_upper"], row["ci80_upper"])
         * Return clipped row dict.

         * Apply reflate_to_nominal to get point_estimate_nominal.

         * Return DataFrame sorted by segment, year.
